## 📦 Requirements

This project uses the following Python libraries:

### Core Data Science & ML
- `numpy` - Numerical computing
- `pandas` - Data manipulation and analysis
- `scikit-learn` - Machine learning algorithms
- `tensorflow` / `keras` - Deep learning
- `torch` - PyTorch neural networks

### NLP & Text Processing
- `nltk` - Natural language processing
- `spacy` - Advanced NLP
- `transformers` - BERT and transformer models

### Visualization
- `matplotlib` - Data visualization
- `seaborn` - Statistical visualizations
- `plotly` - Interactive plots

### Utilities
- `requests` - HTTP requests
- `python-dotenv` - Environment variables
- `jupyter` - Interactive notebooks

### Installation

```bash
pip install numpy pandas scikit-learn tensorflow keras torch nltk spacy transformers matplotlib seaborn plotly requests python-dotenv jupyter
```

Or install from requirements.txt (if available):
```bash
pip install -r requirements.txt
```
